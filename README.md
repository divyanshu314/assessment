# AxyLtd

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 10.0.1.
 
## Prerequisites

Node.js installed.

## Setting up project

Run `npm install - g @angular/cli` in node cmd and navigate to project directory. Install the packages described in the package.json using `npm install` 

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/` to use the app. The app will automatically reload if you change any of the source files.
